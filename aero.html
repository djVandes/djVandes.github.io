Aerospace Projects

DIVA - Debugging aI Verilog Agents

Designing and debugging Verilog RTL Cryptographic algorithms using a system of interacting LM agents locally hosted on LMStudio. Debugging process compares outputs problem solutions from Ollama, Deepseek, and Claude to rank and implement highest ranked solution.

Presenting the project proposal at the 2025 RENEW Conference in Boston, MA in April!

UGLIER - UVM Guideline Linter Extended Ruleset

Using LMs hosted on AWS to generate Verilog linting rulesets. Project included regex generation, debugging, and command implementation to expand upon current lint guidelines.

Maintenance Drone Corporation AI Damage Detection

Using AI Image Processing identify roof damage from drone shots we gathered of Community Housing. Projects include drone flights, annotating and testing training data, and debugging report generators.

Ghost - Search History Obfuscator

Developping a firefox extension to query random searches to obfuscate datacollection in response to ego searches and personally related prompts. Tasks include an AI random search generator, search history database for reference, and debugging the extension UI.
